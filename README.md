# 🦷 Clínica Odontológica

Sistema de gestão para clínicas odontológicas, desenvolvido para agilizar o agendamento de consultas e o cadastro de pacientes e profissionais com possibilidade de apliação para outras áreas da tecnologia.

## 🚀 Tecnologias Utilizadas

* **Frontend:** React (JavaScript), HTML e CSS
* **Backend:** Java com Spring Boot
* **Banco de Dados:** Relacional (via Docker) e Flyway para migrations
* **Infraestrutura e Armazenamento:** Docker, Docker Compose e Cloudinary

## ⚙️ Como Configurar e Rodar o Projeto

### 1. Pré-requisitos
Certifique-se de ter instalado em sua máquina:
* [Git](https://git-scm.com)
* [Docker](https://www.docker.com/) e [Docker Compose](https://docs.docker.com/compose/)

### 2. Clonando o Repositório
```bash
git clone [https://github.com/grsantos56/clinica-odontologica.git](https://github.com/grsantos56/clinica-odontologica.git)
cd clinica-odontologica
```

### 3. Lembre-se de alterar as credenciais no application.yml e docker-compose
